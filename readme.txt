Эта программа была написана, для удобства заполнения базы данных в формате ObjID, UserID, Username, Flag.
В дальнейшем планируется добавить визуальный интерфейс.


Управление осуществляется через цифры.

Вы можете добавить список имен из txt по правилу одно ФИО 1 строка.
Можете добавить по 1 пользователю вручную.
Очистить текущую коллекцию.
